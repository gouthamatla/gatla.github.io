This notebook is just exploring RNA-Seq and genetic data and map QTLs using tensorQTLs.

This assumes that you have basic knowledge in python and takes you through some useful snippets to handle data using pandas library.

You need to have installed pandas, seaborn, scikit-learn and tensorQTL. You also need plink to convert the VCF to plink format. These can be easily installed using conda with python 3.6 for example.

eQTL analysis is basically associating a genetic variant with a gene expression given that you have measured gene expression and genotypes (Arrays or by WGS) from several individuals. Usually it requires a decent sample size around 70 samples. You can still call QTLs with sample size as low as 20 samples, but that might not have statistical power. GTEx establushed a sample size of 70 to have decent results from QTL analysis.

What you need is basically:

A Normalized Gene expression file.

Correspondin genotypes.

The basic workflow is :

1. Gene expression data is normalized (TPM/FPKM etc).

2. The normalized gene expression data might have lot of variation, both technical and biological. As we are intrested only on variation caused by genetic effects, we will remove any non-genetic variation.

3. This can be achived if you have all the metadata (covariates) that might have induced variation in samples. As we usually dont have, we could just use Principle componenents that capture the variation across samples and use the PCs as covariates while calling QTLs. There are other methods that capture technical and unwanted variation like PEER.

4. Once we have normalized gene expression and PCs calculated, we can proceed for QTL association.

5. As with the gene expression, the genotypes also have some technical variation that might interfere with the QTL association. We could calculate the PCs on genotypes, referred as genetic PCs and merge with gene expression PCs and perform QTL association.

We will be using gene-expression and genetic data from 462 lymphoblastoid cell lines RNA-Seq data from 1000 Genomes project. Both the gene expression and genotypes ara available without dbGap access.

The data is available at https://www.ebi.ac.uk/arrayexpress/experiments/E-GEUV-1/

Abstract says:
lymphoblastoid cell lines of 462 individuals from the 1000 Genomes Project, the first uniformly processed high-throughput RNA-sequencing data from multiple human populations with high-quality genome sequences

Lets get the gencode v19 annotations as mentioned in original manuscript.¶

Lets download the annotations and used a simple awk command to get the TSS information

wget ftp://ftp.ebi.ac.uk/pub/databases/gencode/Gencode_human/release_19/gencode.v19.annotation.gtf.gz

zcat  gencode.v19.annotation.gtf.gz | grep -v "^#" | awk -v OFS="\t" '{ if ($3=="gene") print $1,$4,$5,$7,$10 }' | awk -F"\t" -v OFS="\t" '{ gsub("\"","",$5); gsub(";","",$5); if ($4=="+") {print $1,$2,$2,$5,".",$4} else { print $1,$3,$3,$5,".",$4}}' | sort -k1,1 -k2,2n | uniq > gencode.v19.annotation.gtf.tss.bed
